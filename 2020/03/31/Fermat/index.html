费马小定理 快速幂



Description：输入n，计算2的n次方的值。n为整数，且1<=n<=1000000000。由于答案过大，答案对1000000007取模。

费马小定理：若p为素数，\(\gcd(a,p)=1\)，则\(a^{p-1}\equiv{1}\pmod{p}\)
快速幂：将取幂的任务按照指数的二进制表示来分割成更小的任务
模的性质：\((a+b)\pmod{p}=(a\pmod{p}+b\pmod{p})\pmod{p}\)
\((a\times{b})\pmod{p}=(a\pmod{p}\times{b}\pmod{p})\pmod{p}\)

#include< iostream>
#define mod 1000000007
using namespace std;

typedef long long LL;
LL quickPower(LL, LL);
int main() {
	string t;  //因为n过大所以用字符串存储
	cin >> t;
	LL mod1 = mod - 1;
	LL n = t[0] - '0';
	for (int i = 1; i < t.length(); i++) {
		n = (n * 10 + (t[i] - '0')) % mod1;
	}
	cout << quickPower(2, n);
	return 0;
}

LL quickPower(LL x, LL n) {
	LL result = 1;
	while (n > 0) {
		if (n & 1) {
			result = result * x % mod;
		}
		x = x * x % mod;
		n = n >> 1;
	}
	return result;
}

Postscript：\(1\times10^9{+}7\) 是个质数，与\(2\)互质，由费马小定理，\(2{1\times10{9}+6}\equiv1\pmod{1\times10{9}+7}\)，\(n=x\times(1\times10{9}+6)+k\)，再由模的性质\((2{x\times({1\times10{9}+6})}\times2{k})\pmod{1\times10{9}+7}=(1\times2{k})\pmod{1\times10{9}+7}\)，将\(n\)降低为\(k\)，为了防止溢出每一步都进行模运算，同时用快速幂降低时间复杂度。
