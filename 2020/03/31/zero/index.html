阶乘末尾零的个数



Description：给定一个正整数N，求N！末尾0的个数。

#include< iostream>
using namespace std;
int main() {
    long long n,ans = 0;
    cin >> n;
    while (n / 5) {
        ans += n / 5;
        n /= 5;
    }
    cout << ans;
    return 0;
}

Postscript：N！的末尾有0必然是产生了10，2 5 = 10，因为每两个数就能产生一个2的倍数，而每五个数才能产生一个5的倍数，故5的个数比2多，所以只需计算5的倍数的个数就可得出结果，每隔5个数产生一个5的倍数，每隔25个数产生一个25的倍数，而25 = 5 5，依此类推，故无尽除以5的倍数。
