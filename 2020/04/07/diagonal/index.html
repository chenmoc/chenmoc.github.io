对角线交点



\(Description:\)对于一个\(n\)个顶点的凸多边形，它的任何三条对角线都不会交于一点，请求出\(n\)个顶点的凸多边形对角线交点的个数。

#include<iostream>
using namespace std;

int main() {
	int n;
	cin >> n;
	unsigned long long ans;
	ans = (n - 3) * (n - 2) / 2 * (n - 1) / 3 * n / 4;
	cout << ans;
	return 0;
}

\(Postcript:\)凸多边形对角线条数为\(n\times(n-3)/3\)，但是这题和对角线条数并没有关系。首先不会有三条对角线交于一点，所以过某一点对角线交点有且只能有两条对角线，而这两条对角线实际上是确定了四个顶点，因此我们只需要求出四个顶点的搭配即可得到对角线交点的个数，即\(C_{n}^{4}\)，用\(unsigned\ long\ long\)类型储存结果和用代数规律防止结果爆掉(懒得写高精)。
